1. open "C- Compiler.exe" (or "C- Compiler - debug version.exe" for 
                           debug version);
2. choose your C-files;
3. COMPILE->TRACE: generate trace files;
   COMPILE->BUILD->P-CODE: generate p-codes;
   COMPILE->BUILD->ASM: generate asm codes, afterwards, use "ml"
                        (make and link for MASM 6.0) to generate binary files.
